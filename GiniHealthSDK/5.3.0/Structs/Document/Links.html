Links

public struct Links

Links to related resources, such as extractions, document, processed or layout.

  • init(giniAPIDocumentURL:)

    An initializer for a Links structure if you are receiving a customized JSON structure from your proxy backend. For this particular case all links will be pointed to the document’s link.

    Note

    Custom networking only.

    Declaration

    Swift

    public init(giniAPIDocumentURL: URL)

    Parameters

    giniAPIDocumentURL

    The document’s link received from the Gini API. This must be the same URL that you received in the Location header from the Gini API. For example “https://pay-api.gini.net/documents/626626a0-749f-11e2-bfd6-000000000000”.
