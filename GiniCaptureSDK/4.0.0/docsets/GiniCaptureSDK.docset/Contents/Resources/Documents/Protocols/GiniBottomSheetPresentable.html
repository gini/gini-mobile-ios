GiniBottomSheetPresentable

public protocol GiniBottomSheetPresentable

A protocol defining the interface for view controllers that can be presented as bottom sheets.

  • shouldShowDragIndicator

    A Boolean value indicating whether a drag indicator (grabber) should be shown at the top of the sheet.

    Declaration

    Swift

    var shouldShowDragIndicator: Bool { get }
  • shouldShowInFullScreenInLandscapeMode

    Determines whether the sheet should be visible on full screen in compact modes.

    Declaration

    Swift

    var shouldShowInFullScreenInLandscapeMode: Bool { get }

    Return Value

    true if the sheet should be displayed in full screen when in landscape mode, false otherwise.

  • configureBottomSheet(shouldIncludeLargeDetent:)

    Configures the bottom sheet presentation for the view controller.

    • shouldIncludeLargeDetent: A Boolean that determines whether a .large() detent should be included. If false, a .medium() detent will be used instead. Defaults to false.

    Declaration

    Swift

    func configureBottomSheet(shouldIncludeLargeDetent: Bool)
  • updateBottomSheetHeight(to:)

    Updates the bottom sheet’s height using a custom detent.

    • height: The desired height for the bottom sheet.

Available where `Self`: `UIViewController`
  • configureBottomSheet(shouldIncludeLargeDetent:) Default implementation

    Default Implementation

    Configures the view controller to be presented as a bottom sheet.

    • shouldIncludeLargeDetent: If true, the sheet will use a .large() detent. Otherwise, a .medium() detent is used. On iOS 13–14, this falls back to .pageSheet.

    Declaration

    Swift

    func configureBottomSheet(shouldIncludeLargeDetent: Bool = false)
  • presentAsBottomSheet(from:animated:completion:) Extension method

    Presents this view controller as a bottom sheet with proper accessibility support.

    This method configures the view controller as a bottom sheet and presents it modally. After presentation, it automatically hides the presenting view from accessibility, ensures the bottom sheet captures all accessibility focus, and moves VoiceOver focus to the presented content.

    • presenter: The view controller that will present this bottom sheet.
    • animated: Whether to animate the presentation. Defaults to true.
    • completion: An optional closure to execute after the presentation and accessibility configuration are complete.

    Declaration

    Swift

    func presentAsBottomSheet(from presenter: UIViewController,
                          animated: Bool = true,
                          completion: (() -> Void)? = nil)
  • updateBottomSheetHeight(to:) Default implementation

    Default Implementation

    Dynamically updates the bottom sheet height using a custom detent.

    On iOS 16+, it defines and applies a custom detent with the given height.

    • height: The target height for the bottom sheet.
